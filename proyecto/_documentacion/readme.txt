Este proyecto va constatar en crear un sistema de gestion para venta de libros online(WEB).
Este proyecto cuenta con un sitema de gestion(backen) y un sitio web.
El sitio y el backen van a estar construido principalmente con PHP y Mysql para todo lo que es el funcionamiento.
Como parte interacion del usuario vamos a usar HTML5, CSS, y JavaScript con el framework Materialize (https://materializecss.com/) 
El sistema corre bien con PHP 7.4.19, MySQL 5.7.33, Apache 2.4.47

Es compatible con php 8.1.33 en adelante. 
Es compatible con MySQL 5.0 en adelante.
Es compatible con MariaDB 10
